엄켓몬 규칙

1. 기본 — 첫 글자를 "엄"으로 교체 (연속 반복 시 모두 교체)
피카츄 → 엄카츄 · 기기어르 → 엄엄어르
2. 메가진화 — "메가" + 엄켓몬 이름
메가리자몽X → 메가엄자몽X · 메가루카리오 → 메가엄카리오
3. 거다이맥스 — "엄다이맥스" + 엄켓몬 이름
거다이맥스 피카츄 → 엄다이맥스 엄카츄
4. 원시회귀 — "원시" + 엄켓몬 이름
원시가이오가 → 원시엄이오가 · 원시그란돈 → 원시엄란돈
5. 리전폼 / 폼체인지 — 기본과 동일
알로라 라이츄 → 엄이츄 · 기라티나(오리진) → 엄라티나
* 메가 / 거다이맥스 / 원시는 띄어쓰기 있어도 없어도 정답
* 영어 대소문자 구분 없음
